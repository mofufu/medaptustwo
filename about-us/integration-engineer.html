Integration Engineer

Company Overview

MedAptus products have been the gold standard when it comes to hospital and physician point-of-care revenue cycle management for years. Now MedAptus is expanding their product suite to include a new suite of products while continuing to build out more features of their product line.

Position Overview

The Integration Engineer will be responsible for complete project lifecycle of MedAptus healthcare system interfaces. This includes planning, design, development, testing, final implementation and support. Strong communication skills are required as the Integration Engineer will be responsible for project coordination, specification creation and review, as well as working with customers/vendors in identifying and solving interface challenges. This role offers structured install procedures and design however relies heavily on the Integration Engineers analytical and problem solving skills.

Required Education, Skills, Knowledge and Experience

  1. BS in Computer Science, MIS, or equivalent course work
  2. Ability to work in a team environment that includes cross department representatives from both development and deployment staffs
  3. Ability to work with and coordinate discussions with Hospital Administrators, Billing Managers, and IS staff simultaneously during an interface design
  4. 2-5 years experience in project management and design for hospital/clinic based interfaces
  5. 2-5 years experience in written and oral communicating with software vendors during an interface deployment
  6. 2+ years technical software support
  7. Full understanding of HL7 2.x Standard specifications, especially but not limited to ADT, SIU, DFT. HL7 FHIR experience optional but is a plus.
  8. Experience with data evaluation in a variety of formats: XML, Relational DB table, delimited, name/value pairs, fixed length
  9. Relational DB and query language experience, SQL stored procedure development
  10. Understanding HIS or Practice Management Systems usage/functionality
  11. Mirth experience is a plus

Software Skills

  1. MS SQL and MySQL
  2. JavaScript
  3. Understanding network communications using TCP/IP and FTP
  4. Python a plus
  5. Java a plus
  6. Utilizing Interfaceware Chameleon/Iguana a plus
  7. MS Word / Excel / Visio / Project
